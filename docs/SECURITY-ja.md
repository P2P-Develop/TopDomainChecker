# セキュリティーポリシー

## サポート済みバージョン

GETリクエストはhttpsを用いたSSL暗号化通信を介しています。

| バージョン | サポート済み         |
| -------- | ------------------ |
| 2.x      | :white_check_mark: |
| 1.x      | :x:                |

## 脆弱性を発見したら

プログラムに何らかの脆弱性を発見したら、[Github Issues](https://github.com/P2P-Develop/TopDomainChecker/issues) にて\[security\]ラベルを付与して立ててください。
